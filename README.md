### About
<mark>ResLPR</mark> is a benchmark for LiDAR-based place recognition under adverse weather. **ResLPRNet**, a lightweight network, restores corrupted LiDAR scans using wavelet transform. It helps pretrained models enhance the robustness of place recognition in bad weather in a **plug-and-play** manner.
